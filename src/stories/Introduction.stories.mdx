# Sobre

Esse projeto foi feito com o intuito de colocar em prática a metodologia BEM, que é uma metodologia de desenvolvimento de software que visa a simplificação de código e a organização de código.
Também foi envolvido o uso de Storybook para facilitar a visualização dos componentes.
E por fim React-hook-forms, que é um componente de formulário de forma mais simples.
